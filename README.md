Тестовое задание для 3snet

Автоматизация тестирования web-страницы с генератором календаря спортивных событий
https://dev.3snet.info/eventswidget/

Скачайте проект 

Установите зависимости
```
sudo apt-get install nodejs npm
```
В папке с проектом
```
npm install @playwright/test
npx playwright install
```
Тесты разделены на группы по тегам.
@smoke
@critical-path
@extended

Запуск тестов. Из папки с проектом
```
npx playwright test --grep "@smoke|@critical-path"
```
Текущее состояние тестов

Smoke тесты, связанные с главной фичей, генератором календаря, работоспособны.
Часть smoke и critical-path тестов, связанных с навигацией по web сервису 3snet, падает из-за отсутствия равнозначной поддержки главного сайта и стенда с тестовым заданием. 
На данный момент extended тесты не реализованы
